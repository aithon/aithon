void aiAnalogInit ( void  )

Analog function initialization.

This function sets up and starts the ADC conversions for the various analog inputs on the Aithon board. This includes all 8 analog input pins, the motor current sensors, and the main and servo input voltages.

Note
This function is called by aiInit() and does not need to be called again by the user after aiInit() is called.
uint16_t aiAnalogInput ( AnalogPin  pin)

Get value of an analog input.

This function returns the average of the last 100 ADC readings on an analog input pin.

Parameters

Returns
